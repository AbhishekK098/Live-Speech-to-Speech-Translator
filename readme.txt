Speech Translator GUI App
Features:
- Transcribe speech using Whisper
- Auto detect language
- Translate into Hindi, Spanish, French
- Speak translations in natural female voices using Edge TTS
- Save transcripts with timestamps
- Simple Tkinter GUI
Usage:
1. Run `pip install -r requirements.txt`
2. Run app with: `python translator_gui.py`
3. Press "Start Listening" and speak.
4. Say "stop" to end listening.
To create .exe (Windows):
Run: pyinstaller --onefile --windowed translator_gui.py
